# Assets Folder

This folder contains images, screenshots, and other media assets for the project documentation.

## Screenshots to Add

For a complete portfolio project, consider adding:

1. **streamlit-ui.png** - Screenshot of the Streamlit web interface
2. **chat-example.png** - Example conversation with source citations
3. **architecture-diagram.png** - System architecture diagram (optional)
4. **notebook-demo.png** - Screenshot of the Jupyter notebook in action

## How to Add Screenshots

1. Run the Streamlit app: `streamlit run app.py`
2. Take screenshots of the interface
3. Save them in this folder
4. Reference them in the main README.md using:
   ```markdown
   ![Description](assets/filename.png)
   ```

## Recommended Image Sizes

- Screenshots: 1200-1600px wide
- Diagrams: 800-1200px wide
- Format: PNG for screenshots, SVG for diagrams (if possible)
